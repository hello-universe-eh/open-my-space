데이터 동기화 모드

로컬 데모는 브라우저 내부 저장소(LocalStorage)를 사용하여 나에게만 데이터가 보입니다. 모든 교인이 함께 사용하려면 실시간 동기화 모드로 설정해 주세요.

Supabase API 설정

Supabase(supabase.com) 가입 후 무료 프로젝트를 생성하여 아래 정보들을 입력하세요. 해당 정보는 브라우저에 안전하게 저장됩니다.

Supabase 테이블 생성 SQL

Supabase의 SQL Editor에 아래 스크립트를 복사하여 실행하면 데이터베이스 세팅이 끝납니다. 

-- 1. spaces 테이블 생성
CREATE TABLE spaces (
  id uuid DEFAULT gen_random_uuid() PRIMARY KEY,
  created_at timestamp with time zone DEFAULT timezone('utc'::text, now()) NOT NULL,
  host_name text NOT NULL,
  space_name text NOT NULL,
  location text NOT NULL,
  capacity integer NOT NULL,
  fee text NOT NULL,
  parking_info text DEFAULT 'none' NOT NULL,
  description text
);

-- 2. registrations 테이블 생성
CREATE TABLE registrations (
  id uuid DEFAULT gen_random_uuid() PRIMARY KEY,
  created_at timestamp with time zone DEFAULT timezone('utc'::text, now()) NOT NULL,
  space_id uuid REFERENCES spaces(id) ON DELETE CASCADE,
  user_name text NOT NULL,
  user_phone text NOT NULL,
  party_size integer DEFAULT 1 NOT NULL,
  parking_required boolean DEFAULT false NOT NULL,
  UNIQUE (user_name, user_phone)
);

-- 3. Row Level Security(RLS) 비활성화 (누구나 읽기/쓰기가 가능하도록 임시 해제)
ALTER TABLE spaces DISABLE ROW LEVEL SECURITY;
ALTER TABLE registrations DISABLE ROW LEVEL SECURITY;
